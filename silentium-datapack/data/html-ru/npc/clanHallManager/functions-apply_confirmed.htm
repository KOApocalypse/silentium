Функция Холла клана была активированна.
Деньги будут автоматически сниматься из хранилища клана по мере окончаниия времени их действия. Если на Вашем клановом хранилише будет недостаточно денег то услуги будут анулированны.
Проверить время действия для каждой функции можно в меню "Управление Холлом".
